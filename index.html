Data Annotation

Upload a CSV file to start annotating. You can also upload a partially annotated CSV file to continue where you left off.

This website does not store your data. Be sure to download your annotated data before closing or reloading the page.

The first column should contain the data, and subsequent columns should be category names.


Example CSV format:

data cat1 cat2 cat3 ...
