Data Annotation

Upload a CSV file to start annotating

The first column should contain the data, and subsequent columns should be category names.


Example CSV format:

data cat1 cat2 cat3 ...
"This is the first data sample"
"Another data sample here"
"Yet another data sample"

Instructions

- Read the text and select appropriate categories by clicking on the buttons.

- Navigate using the "Previous" and "Next" buttons, or use keyboard shortcuts:

  A for previous, D for next.


- Don't forget to save your annotations when you're done!
